# Creating a Blog Post

SimplCMS provides an easy-to-use interface to create and manage blog posts.

## Create a New Blog Post

To create a new blog post:

1. Navigate to your admin dashboard:

```
http://your-domain.com/admin/posts/new
```

2. Use the intuitive wizard to add your blog content, including:
   - Title
   - Content body (Markdown supported)
   - Categories and tags
   - Featured images and media

## Save as Draft or Publish

You can save your blog post as a draft or publish it immediately:

- Click **"Save as Draft"** to save your post without making it public.
- Click **"Publish Post"** to immediately make your post public.

## Editing Existing Posts

You can easily manage your existing blog posts by visiting:

```
http://your-domain.com/admin/posts
```

From here, you can edit drafts, publish them, or update previously published content.
